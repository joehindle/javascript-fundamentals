# javascript-fundamentals

This repository is where I learn and practise core JavaScript fundamentals.

The aim is to build a strong foundation in the language through small exercises, simple problems, and regular practice. This repo is focused on understanding JavaScript itself rather than web development, frameworks, or external tools.

## Topics Covered

- Variables and data types
- Operators
- Conditionals
- Loops
- Functions
- Arrays
- Objects
- Strings
- Scope
- Basic problem-solving

I created this repository to:

- improve my understanding of JavaScript fundamentals
- practise writing clean and readable code
- strengthen problem-solving skills
- track my progress as I learn
